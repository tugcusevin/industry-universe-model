# Industry Universe Model – Theoretical Framework / Endüstri Evreni Modeli – Teorik Çerçeve

## Overview (EN)
The **Industry Universe Model** is a layered conceptual framework that explains industrial transformation as an **evolutionary structure** across three paradigms:

- **Lean Manufacturing** → Operational **Stability**
- **Industry 4.0** → Digital **Capacity**
- **Industry 5.0** → Normative & Regenerative **Value**

Rather than treating these paradigms as isolated stages, the model frames them as complementary layers of a single socio-technical production system.

---

## Model Architecture (EN)

### 1) Layered Evolution
The model is built on three vertically connected layers:

1. **Lean Manufacturing (Stability Layer)**  
   Focus: process discipline, waste reduction, standardization, flow stability.

2. **Industry 4.0 (Capacity Layer)**  
   Focus: connectivity, data infrastructure, cyber-physical integration, analytics-driven optimization, autonomy.

3. **Industry 5.0 (Value Layer)**  
   Focus: human-centric design, sustainability, ethics, resilience, and regenerative value creation.

### 2) Five Transformation Axes
The model is analyzed through five horizontal axes that cut across all layers:

1. **Operational Efficiency**
2. **Digital Transformation**
3. **Autonomy & Intelligent Systems**
4. **Human-Centricity**
5. **Sustainability**

Each axis may appear differently at each layer (e.g., as operational practices in Lean, technological capabilities in Industry 4.0, and normative goals in Industry 5.0).

### 3) Core Transition Logic
The model’s core logic can be summarized as:

- **Function transition:** Stability → Capacity → Value  
- **Nature transition:** Operational → Technological → Normative  
- **Level transition:** Micro → Meso → Macro

---

## Concept Classification Logic (EN)
Each concept in the dataset is classified using:

- **layer**: Lean / Industry4 / Industry5  
- **primary_axis**: one dominant axis (Operational, Digital, Autonomy, Human, Sustainability)  
- **secondary_axes**: optional cross-axis relevance  
- **function**: Stability / Capacity / Value  
- **nature**: Operational / Technological / Normative / Hybrid  
- **level**: Micro / Meso / Macro  

This structure supports both academic clarity (via primary classifications) and graph-based analysis (via secondary axes).

# Genel Bakış (TR)
**Endüstri Evreni Modeli**, sanayi dönüşümünü üç paradigma üzerinden **evrimsel ve katmanlı** bir yapı olarak açıklayan kavramsal bir çerçevedir:

- **Yalın Üretim** → Operasyonel **Stabilite**
- **Endüstri 4.0** → Dijital **Kapasite**
- **Endüstri 5.0** → Normatif ve Rejeneratif **Değer**

Bu yaklaşım paradigmaları birbirinden kopuk aşamalar olarak değil, aynı üretim sisteminin birbirini tamamlayan katmanları olarak ele alır.

---

## Model Mimarisi (TR)

### 1) Katmanlı Evrim
Model üç dikey katman üzerine kuruludur:

1. **Yalın Üretim (Stabilite Katmanı)**  
   Odak: süreç disiplini, israf azaltma, standartlaştırma, akış stabilitesi.

2. **Endüstri 4.0 (Kapasite Katmanı)**  
   Odak: bağlantılılık, veri altyapısı, siber-fiziksel entegrasyon, analitik tabanlı optimizasyon, otonomi.

3. **Endüstri 5.0 (Değer Katmanı)**  
   Odak: insan merkezli tasarım, sürdürülebilirlik, etik, dayanıklılık ve rejeneratif değer üretimi.

### 2) Beş Dönüşüm Ekseni
Model, tüm katmanları kesen beş yatay eksen üzerinden okunur:

1. **Operasyonel Verimlilik**
2. **Dijital Dönüşüm**
3. **Otonomi ve Akıllı Sistemler**
4. **İnsan Merkezlilik**
5. **Sürdürülebilirlik**

Her eksen katmanlara göre farklı biçimde görünür (ör. Yalın’da operasyonel pratik, 4.0’da teknolojik yetkinlik, 5.0’da normatif hedef).

### 3) Temel Geçiş Mantığı
Modelin omurgası şu dönüşümlerle özetlenir:

- **Fonksiyon dönüşümü:** Stabilite → Kapasite → Değer  
- **Doğa dönüşümü:** Operasyonel → Teknolojik → Normatif  
- **Seviye dönüşümü:** Mikro → Meso → Makro  

---

## Kavram Sınıflandırma Mantığı (TR)
Veri setindeki her kavram şu alanlarla sınıflandırılır:

- **layer**: Lean / Industry4 / Industry5  
- **primary_axis**: baskın eksen (Operational, Digital, Autonomy, Human, Sustainability)  
- **secondary_axes**: opsiyonel kesişim eksenleri  
- **function**: Stability / Capacity / Value  
- **nature**: Operational / Technological / Normative / Hybrid  
- **level**: Micro / Meso / Macro  

Bu yapı, akademik sunumda açıklığı korurken (primary), ağ/graph analizine de imkan verir (secondary).
