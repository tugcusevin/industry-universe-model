# Methodology

## Concept Coding Approach

The Industry Universe Model uses a structured concept classification system to organize industrial transformation concepts.

Each concept is described using the following attributes:

- **layer**
  - Lean
  - Industry4
  - Industry5

- **primary_axis**
  - Operational
  - Digital
  - Autonomy
  - Human
  - Sustainability

- **secondary_axes**
  Additional axes that intersect with the primary dimension.

- **function**
  - Stability
  - Capacity
  - Value

- **nature**
  - Operational
  - Technological
  - Normative
  - Hybrid

- **level**
  - Micro
  - Meso
  - Macro

---

## Classification Logic

The model follows a layered transformation logic:

Lean Manufacturing → Stability Layer  
Industry 4.0 → Capacity Layer  
Industry 5.0 → Value Layer

Concepts are positioned according to their dominant role in industrial transformation.

---

## Data Structure

Concepts are stored in:

- JSON format (for structured datasets)
- CSV format (for analytical processing)

The schema file defines the valid structure for concept entries.
